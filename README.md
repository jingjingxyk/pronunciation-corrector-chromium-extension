# pronunciation-corrector-chromium-extension

> 为[`在线英语发音纠正`](https://github.com/songquanpeng/pronunciation-corrector.git) 实时 显示 单词 释义的 chromium 扩展

## 使用说明

- 点击第一列，调用的是搜索框页面
- 点击第二列和第三列，调用的播放音频功能
